## Blink code
> Paste code inside USER region
```c
 /* Infinite loop */
  /* USER CODE BEGIN WHILE */
  while (1)
  {
	  HAL_GPIO_TogglePin(GPIOA, GPIO_PIN_5);
	  HAL_Delay(1000);
    /* USER CODE END WHILE */

    /* USER CODE BEGIN 3 */
  }
  /* USER CODE END 3 */
```

#### build
Project > Build Project

#### Run and Debug
Run > Debug As > STM32 MCU C/C++ Application

- Open with VSCode
